Some Stuff I Want You to Build

Make “The Hitchhiker’s Guide to the Galaxy” Become True

Wikipedia already is a great encyclopedia but the Hitchhiker’s Guide to the Galaxy is way more entertaining because it presents the content with videos.

To build a encyclopedia as big as Wikipedia everyone should be able to contribute. Therefore there should be some tools available to easily produce great animated content. It would be even better if you could use the hole power of the web and create an interactive experience with tools like Mozilla’s Popcorn Maker (https://popcorn.webmaker.org/).
