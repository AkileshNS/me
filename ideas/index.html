Startrek-like Computer

Actually it is pretty easy to enable your own computer to talk with you. Checkout the Web Speech APi: https://www.google.com/intl/en/chrome/demos/speech.html.

The voice recognition is pretty good already.

Now you only have to setup hubot (http://hubot.github.com/) and you have a really big palette of command available: https://hubot-script-catalog.herokuapp.com/.

It shouldn’t be that hard to write a little web app sending requests to hubot.

Unfortunately Web Speech API only available in Chrome yet. Would be even more fun on mobile.

btw: Did you ever clicked the little microphone in the google search field (only in Chrome)? Just tell her “make me a sandwich” :)

Send Mail to an Email Address

It would be awesome if you could just put an e-mail address on a postcard or a letter instead of a real address.

You would not have to remember the address of someone if you want to send him something.

The post office could just send an e-mail to the given e-mail address and ask for a real address to deliver the mail to.

This way you could send postcards to people back home from your holidays without asking them for their address and it also would be useful the other way around: you don’t have to give someone your real address to get mail.

Furthermore it’s also handy for people who travel a lot: they could just choose a new address for every single delivery.

The problem is that you only can realize this idea if you are a mail delivery company.

Excel Replacement in a JSON / NoSQL Way

Excel has been the only choice for a lot of office work for the last years.

But maybe some tasks Excel is used for could be accomblished way easier if there would be a more flexible tool that would not force you to use tables for everything.

This would be like nosql technologies as alternative to sql databases.

The data could be stored in json instead of the annoying .csv format and macros could be written in a modern scripting language like Javascript instead of Visualbasic.

I think there is a lot of things that could be improved in Excel and its a pitty that even if so many people’s work life depends on this tool there is not much effort put into improving it.

Start Startup in a Developing Country

Building a startup in a country with nearly no infrastructure has some challenges. It can be difficult to find good employees and paying costumers. But it has some advantages too.

The sallaries are low and there is not much competition.

You could start the project by collecting donations with the help of platforms like kickstarter.com and betterplace.org. People could donate their old Hardware too.

You shouldn’t search for employees but for local talents instead. They do not need experience in software development. Search for young, smart people instead.

The first phase of the project would be to teach your new emploees in the areas they’re most talented in. And as soon as you have a small team you could start earning money.

If there is now interest for software developers by local businesses you still can do work for international businesses or be creative and invest time in platforms like http://themeforest.net/.
